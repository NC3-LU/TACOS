Privacy policy

Security made in Lëtzebuerg g.i.e. built the TACOS application as a open source software. This service is provided by Security made in Lëtzebuerg g.i.e. at no cost and is intended for use as is.

This page is used to inform website visitors regarding our policies with the collection, use, and disclosure of personal information if anyone decided to use our service.

The terms used in the privacy policy available at tacos.cases.lu and in the mobile application have the same meanings.

Information collection and use

This service is not collecting any personally identifiable information. The main goal is to allow the user to detect spam phone numbers. Incoming phone numbers from the smartphone are locally filtered, locally hashed (with the algorithm SHA-512), then locally compared to already reported spam from our database. For the filtering of the call logs the application requires the permission to read the call logs (android.permission.READ_CALL_LOG) of the device.

If you choose to use our service, then you agree to the manual collection and use of information in relation with this policy. The information that we collect (hashed value of phone number) will be used to improve this service.

By manual collection it is important to understand that you will always decide which phone number you want to report.

Log data

We do not collect log data from the device.
